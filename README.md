# 🎮 Pixel Play

> A modern, high-performance web arcade platform for classic and casual games — right in your browser.

---

## ✨ Overview

**Pixel Play** is a sleek, responsive web arcade built with modern technologies. It brings together a collection of retro classics and logic-based games in a clean, cyberpunk-inspired interface.

No downloads. No accounts. Just click and play.

---

## 🚀 Features

### 🕹️ Diverse Game Library
Enjoy 15+ games across multiple categories:

- **Arcade Classics**
  - Snake
  - Tetris
  - Breakout
  - Dino Run
  - Flappy Bird

- **Puzzles & Logic**
  - Sudoku
  - Minesweeper
  - 2048
  - Wordle

- **Memory & Skill**
  - Memory Match
  - Whack-a-Mole
  - Tower Blocks
  - Hangman
  - Tic Tac Toe
